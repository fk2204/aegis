- State regulation Tier promotion (Tier 3 → Tier 1) decisions

- OFAC screening, reporting, retention workflow
- AML/CIP cascade obligations

These dossiers are NOT authoritative for:
- Specific funder ISO contract obligations (varies by contract)
- IL Loan Brokers Act applicability to AEGIS (operator phone verification required)
- Anything that depends on AEGIS's specific operational facts I can't see

## When to consult an attorney

Per dossier 15, attorney consult required for:
- MCA reclassification claim or threatened lawsuit
- Cybersecurity incident
- Regulator inquiry from any state AG, NY DFS, CFPB, OFAC, FTC
- Subpoena or preservation letter
- Pivot in business model (direct funding, money transmission, consumer products)
- Before funding first deal in any new state (template review)
- Before signing funder ISO agreements (cascade review)
