# TX — Statutes and citations

**Statute:** HB 700 (2025); TX Finance Code Chapter 398.

**Effective:** 2025-09-01 (disclosure); 2026-12-31 (registration).

**Scope:** MCA only (sales-based). Disclosure threshold ≤ $1,000,000; registration applies regardless of deal size.

**APR:** Not required.

**Broker registration:** Required — TX OCCC, by 2026-12-31, annual renewal by January 31.

**Penalties:** $10,000 per violation. TX OCCC enforces. No PRA.

**Notes:** **AUTO-DEBIT PROHIBITED** without first-priority perfected UCC security interest in the merchant's deposit account — deal-killer for standard MCA. AEGIS hard-declines TX standard ACH-debit MCA via `tx_autodebit_without_first_priority_lien` rule. COJ void in MCA contracts.
